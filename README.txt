This application is build using Angular 5 and ASP.NET WebAPI 2

Before running a solution. make sure that Node.js 9.3.0 is installed. If not, it can be downloaded here: https://nodejs.org/dist/v9.3.0/node-v9.3.0-x64.msi

Running procedure:
1) Open solution in Visual Studio 2017
2) Click on Run
3) Open an Command Prompt at <unpacked arhive path>\SnowSoftwareExercise\SnowSoftwareExercise
4) Type 'npm start' in console
5) Open Google Chrome or Firefox at address localhost:4200
